#  e-Learn

e-Learn is a simple educational web application built using **HTML, CSS, and JavaScript**.
It provides multiple learning-related utilities in one platform, including video learning, quizzes, productivity tools, and mentorship features.

---

##  Features

*  **E-Video** – Access learning videos
*  **E-Book** – Browse digital study material
*  **Quiz** – Test knowledge with interactive quizzes
*  **To-Do List** – Manage study tasks
*  **Mentorship** – Explore guidance features
*  **Dictionary** – Look up word meanings
*  **Tutor Nearby** – Find nearby tutors (UI feature)

---

##  Tech Stack

* HTML
* CSS
* JavaScript

*(Frontend only – no backend integration)*

---

##  Run Locally

1. Clone the repository:

```bash
git clone https://github.com/ShakilUrRehman21/e-learn.git
cd e-learn
```

2. Open `index.html` in your browser.
